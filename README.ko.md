## ✨ 3분 사용 가이드

### 1️⃣ 스크린샷 업로드 (5초)
드래그 앤 드롭으로 앱 스크린샷 업로드

### 2️⃣ AI 자동 분석 (10초)
- 앱 이름, 카테고리 자동 인식
- 주요 기능 추출
- 타겟 오디언스 파악
- 브랜드 컬러 감지

### 3️⃣ 결과 확인 (즉시)
**5가지 스타일 카피 생성:**
1. 💪 **Power**: "Think. Plan. Do." (단순 강렬)
2. 🎁 **Benefit**: "Save 3 hours daily" (혜택 강조)
3. 👥 **Social Proof**: "#1 Productivity App" (사회적 증거)
4. ⚙️ **Feature**: "AI-powered summaries" (기능 설명)
5. 💝 **Emotional**: "Finally, peace of mind" (감성)

**보너스: Twitter 스레드 3개**
- 월요일: 창업 스토리
- 수요일: 기능 소개
- 금요일: 사용자 후기
